🧠 Virtual CPU in Assembly

A custom-built virtual CPU written in x86-64 assembly.
This project simulates how a real processor works internally using a fetch-decode-execute cycle.

---

⚙️ Features

- 16 virtual registers
- 4KB memory
- Program counter (PC)
- Custom instruction execution
- Low-level memory manipulation

---

🔁 CPU Cycle

Fetch → Decode → Execute → Repeat

---

🧪 Example Instructions

LOAD V0, 5
ADD V0, 3
STORE V0, 100

---

🛠 Built With

- x86-64 Assembly (NASM)
- Linux syscalls
